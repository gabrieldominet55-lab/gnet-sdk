## 🧠 How it Works
The bridge uses a **Decoupled Event-Driven Handshake** to teleport your game logic into the Gnet Vault without affecting your local performance.

1. **Handshake**: Encrypted link establishment with Gogi's Gatekeeper.
2. **Snapshot**: Automated packaging of your game logic and assets.
3. **Teleport**: Asynchronous data transfer to the secure Gnet infrastructure.
4. **Live Boot**: Automatic grab and execution by Gogiroy_Jay on Twitch/YT.

## 🛠 Quick Start

### 1. Installation
Drop the `gnet_logic_bridge.py` into your project root.

### 2. Integration
```python

# 🔗 Setup the link
bridge = GnetLogicBridge(
    dev_id="YourName", 
    project_id="EpicProject_v1"
```
