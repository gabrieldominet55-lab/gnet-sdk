# 🚀 Gnet.AI Open SDK
The ultimate tool for indie developers to get global exposure.

## ✨ Why use Gnet SDK?
- **Automatic Export**: Package your game for any platform in one click.
- **Gogi Star Showcase**: Every game exported with this SDK is automatically sent to **Gogiroy_Jay**.
- **Free Marketing**: Gogi will play your game live on Twitch and YouTube, showing it to thousands of fans!

## 📦 Installation
```bash
git clone https://github.com
```

## 🎮 How to use
```python
from gnet_sdk import GnetSDK

# Initialize
sdk = GnetSDK(dev_name="YourName", project_name="SuperLogicGame")

# Export and send to Gogi
sdk.export_and_ship("./my_build_folder")
```
